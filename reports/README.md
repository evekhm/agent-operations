This directory contains the Agent Health Check Reports. These are autonomously generated by the `observability_analyst` agent using BigQuery queries against event views.

### Viewing Generated Reports

To view the generated reports, it is highly recommended to use the [Markdown Viewer](https://chromewebstore.google.com/detail/markdown-viewer/ckkdlimhmcjmikdlpkmbgfkaikojcbjk?hl=en-US&utm_source=ext_sidebar) Chrome Extension.

**Recommended Settings:**
*   **Theme**: Github Wide
*   **Content**: Make sure to enable **`mermaid`** and **`emoji`**
